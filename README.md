# Personal Portfolio Website

A personal developer portfolio website designed to showcase projects, technical skills, and professional resume information.

## Built With

This project is built using the following technologies:
* React.js
* React-Bootstrap
* CSS3
* Node.js
* VS Code

## Features

* Multi-Page Layout: Structured sections across separate routes.
* Easy Styling: Powered by React-Bootstrap and custom CSS variables.
* Fully Responsive: Designed to scale across desktop, tablet, and mobile screens.

## Installation and Setup Instructions

To run this project locally, make sure you have Node.js and Git installed on your machine.

1. Clone the repository down to your local system.
2. Open your terminal inside the project root folder.
3. Install dependencies with legacy peer support by running:
   npm install --legacy-peer-deps
4. Launch the local development server by running:
   npm start

The application will run in development mode. Open http://localhost:3000 in your browser to view it. The page will automatically reload whenever you make edits and save your files.

## Usage and Customization Instructions

To update the website with your own information:
1. Open the project folder in your code editor.
2. Navigate to the "/src/components/" directory.
3. Locate the corresponding files for each section (Home, About, Projects, Resume).
4. Replace the default text strings and data properties with your personal bio, projects, and experiences.
